# Prompt: Auditoria de UX, Core Web Vitals e Retenção

Use este comando quando precisar diagnosticar problemas de usabilidade, lentidão interativa ou alta taxa de rejeição em uma página específica.

```text
Ative a FASE 1 do pipeline com foco prioritário no `[UXArchitect]` e no `[TechSEO]`. 

Quero auditar a seguinte página: [Insira a URL ou descreva a estrutura da página]. O problema atual é: [Ex: o tráfego chega mas ninguém clica no botão / a taxa de rejeição no mobile está alta]. 

Exijo uma análise sobre:
1. Arquitetura de informação visual e escaneabilidade do conteúdo.
2. Atrito de carregamento e potenciais gargalos de Core Web Vitals (especialmente INP - Interaction to Next Paint).
3. Posicionamento de CTAs e ganchos de retenção (Dwell Time).

Aguardando o esqueleto estrutural de melhorias antes de passarmos para a Fase 2.
